# Go-HTMX
A simple web application built with Go and HTMX to demonstrate dynamic content loading and interactivity.

## Features
- Dynamic content loading with HTMX
- Simple and clean user interface
- Easy to extend and customize

## Prerequisites
- Go 1.25 or higher
- HTMX library (included via CDN in the HTML)

## Installation
1. Clone the repository:
    ```bash 
    git clone https://github.com/Nishant-28/go-htmx.git
    cd go-htmx
    go run main.go
    ```
2. Open your web browser and navigate to `http://localhost:8080`
